# 📿 Đại Tạng Kinh Tóm Tắt — OpenClaw Skill

Tra cứu và giải đáp kiến thức **Phật học** dựa trên tài liệu **"Đại Tạng Kinh — Tóm tắt trong 100 trang"** của tác giả **Hạt Bồ Đề (Gosinga)**.

Tóm lược toàn bộ Kinh Phật Nikaya và A-Hàm theo **phương pháp quy nạp toán học** — giúp hiểu nhanh giáo lý Đức Phật.

> ⚠️ Bản tóm lược KHÔNG THAY THẾ Kinh Phật gốc. Đây chỉ là hướng dẫn đọc Kinh Phật nhanh nhất và dễ hiểu nhất.

## ✨ Tính năng

Hỏi OpenClaw bất kỳ câu hỏi nào về Phật học, bot sẽ tra cứu và trả lời dựa trên tài liệu gốc:

- 🔍 **Tứ Thánh Đế, Bát Chánh Đạo, Tứ Niệm Xứ**
- 🧘 **Thiền Jhana, Chánh Niệm, Chánh Định**
- 📖 **Ngũ Uẩn, Duyên Khởi, 12 Nhân Duyên**
- 🌟 **Giác ngộ, Giải thoát, Niết Bàn, Tứ Thánh Quả**
- ☸️ **Lịch sử Phật giáo, Đức Phật, Đại đệ tử**
- 🙏 **Tu tập, Ba La Mật, Hạnh Đầu Đà, Giới-Định-Tuệ**

## 📦 Cài đặt

```bash
# Clone repo
git clone https://github.com/YOUR_USERNAME/dai-tang-kinh-openclaw.git

# Copy vào thư mục skills của OpenClaw
cp -r dai-tang-kinh-openclaw ~/.openclaw/skills/dai-tang-kinh

# Restart OpenClaw
openclaw gateway restart
```
